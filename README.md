# Weather-App
A version of the simple weather app, where it automatically pulls the user's IP and provides the current weather\
This site uses your IP to estimate your location for weather data via ipapi.co nothing is stored

## Initial Setup
run in the terminal:\
   *pip install -r requirements.txt*

Then who test on local localhost simply run:\
   *app.py*
and open up provided link

<br/>

Note that as of 10/7/26 there are some issues with using the ipapi api, this code should work despite this.
